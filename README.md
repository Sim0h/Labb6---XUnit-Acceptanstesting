# Acceptanstester

### Acceptanstest 1: Addition av två heltal

1. **Öppna applikationen.**
2. **Välj räknesättet "Addition".**
3. **Ange det första talet: 5.**
4. **Ange det andra talet: 3.**
5. **Förväntat resultat:** Applikationen ska skriva ut "Resultatet är: 8".
